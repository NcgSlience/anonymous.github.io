                                              # 第一章

                                              1.什么是低压电器？（小题）

                                              交流1200V以下，直流1500V以下的电器叫低压电器

                                              采用电磁式原理的低压电器用的是电磁式低压电器

                                              电磁式低压电器的结构：电池机构，触头，灭弧装置

                                              常用的低压电器：接触器和继电器

                                              2.线圈得电，常开触电常闭，常闭触点常开（小题）

                                              线圈得电，常开触电常开，常闭触点常闭

                                              # 1.1.1接触器

                                              交流接触器和直流接触器中所说的交流和直流是控制回路中（也就是缠绕在线圈上的线圈中所通过的电流）

                                              必考知识点

                                              # 上课时PPT上的问题

                                              交流接触器通相同电压的直流电会发生什么？

                                              我们知道线圈中的电阻在通交流电的时候，电阻本身除了阻值是有电抗的，电流是这样的表达式： 而通了直流电的时候，流过的电流就只有电压除以阻值，所以电流会明显变大，根据功率等于电流的平方乘以电阻和时间， 通电一段事件后，会产生热量，烧坏电阻。

                                              直流接触器通相同电压的交流电会发生什么？

                                              线圈通相同电压的交流电后，电阻会有感抗，会导致电流减小，接触器没有足够的电流启动。

                                              # 1.1.2继电器

                                              保护特性曲线一定要在过载特性曲线的下面，如果超过，器件可能损坏。

                                              # 1.1.3热继电器

                                              热继电器的作用：过流保护。

                                              温度继电器：过温或者过低温保护。

                                              # 1.1.4开关电器

                                              固态继电器采用半导体器件组装而成的一种无触点继电器

                                              低压断路器：俗称空气开关，对电路或电气设备发生严重的短路、严重过载及欠电压等进行保护。

                                              上图中虚线的作用：表示这个图中的所有触头同时闭合，同时断开。

                                              # 1.1.5 熔断器

                                              熔断器的主要由熔断管（或盖或座），熔体及导电部分原件组成，其中熔体为主要组成部分，即是感测元件又是执行元件。

                                              工作原理：基于电流热效应原理以及发热元件热熔断原理设计。

                                              # 经消弧线圈接地

                                              消弧线圈的补偿方式：1.完全补偿、2.欠补偿、3.过补偿方式

                                              电力系统采用过补偿方式而不采用其他补偿方式：

                                              由于系统会临时操作投切某些线路，为避免补偿电感电流与接地故障电流相等，发生谐振，通常广泛采用过补偿方式。

                                              脉动电流的危害及其抑制措施？

                                              危害：脉动部分被的交流分量会增加电机的发热，产生转矩脉动。

                                              抑制措施：（1）增加整流电路的相数，或采用多重化技术

                                              （2）设置足够大的平波电抗器

                                              # DSP原理及应用

                                              -DSP是一种专门用于数字信号处理的微处理器，具有高性能、低功耗和高度可编程的特点。 它的工作原理包括指令获取、解码、数据处理、结果存储和控 制流程等阶段。 DSP在通信、音频、视频、雷达、医疗等领域有着广泛的应用。

                                              # 计算机控制技术

                                              -计算机控制技术》是自动化、自动控制、电气工程及其自动化等电气信息类专业的一门专业主干课程。该课程主要包括计算机控制系统的特点、组成、分类以及计算机 控制系统的发展趋势，计算机控制系统的基本输入输出接口与过程通道，数字控制技术，数字PID控制算法，常规及复杂控制技术，应用程序设计与实现技术，分布式测控网络技术，计算机控制系统设计与实现。

                                              # 可编程逻辑器件及应用

                                              -《可编程器件及应用》是实践性强的技术性基础课程。 以大规模可编程逻辑器件为设计载体，以硬件描述语言为系统逻辑描述的主要表达方式，以计算机、大规模可编程逻辑器件的开 发软件及实验开发系统为设计工具，通过相关开发软件，自动完成用软件方式设计的电子系统到硬件系统的逻辑编译、逻辑化简、逻辑分割、逻辑综合及优化、逻辑布局布线、逻辑仿真、直 至对于特定目标芯片的适配编译、逻辑映射、编程下载等工作，最终形成集成电子系统或专用集成芯片的一门新技术，或称IES/ASIC自动设计技术。
